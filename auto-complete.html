Search for countries

    Simple Fluid and responsive auto-completion search form

    Features

    Not all countries are listed here, these are just sample coutries, use an API in place of array in production

    1. Auto prediction
    2. Up and down arrow keys to select options
    3. Esc key to dismiss form
    4. Smooth fluid height animation
    5. Infinite scrolling if enabled - set data-infinite="true" to enable and "false" to disable
    6. Simple loops for best performance
    7. Automatically recalculate the height depending on its content
